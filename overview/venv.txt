Virtual Environments:

   Create a Virtual Environment by running the venv package,
 which creates a virtual environment named venv.  The first
 venv in the command is the name of the Python virtual
 environment package, and the second is the virtual environment
 name.

$ python3 -m venv venv


   Activate Virtual Environment:

$ source venv/bin/activate
(venv) $ 


   Deactivate Virtual Environment:

$ deactivate
